# 🏙️ CivicAlert — Crowd Issue Reporter

A full-stack civic issue reporting platform where citizens 
can report, track and upvote local infrastructure problems.

## 🚀 Tech Stack
- **Frontend:** React.js, Axios, React Router
- **Backend:** Node.js, Express.js, REST API
- **Database:** MySQL, Sequelize ORM
- **Auth:** JWT, bcryptjs, Role-based Access Control

## ✨ Features
- JWT Authentication with 3 user roles
- Report issues with photo uploads
- Upvoting system for priority issues
- Anonymous reporting for privacy
- Real-time admin dashboard with analytics
- Comment system with official replies
- Filtering, sorting and search

## 🛠️ Setup
1. Clone the repo
2. cd backend → npm install → npm run dev
3. cd frontend → npm install → npm start
